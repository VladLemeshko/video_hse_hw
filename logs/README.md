# Логи выполнения

Эта папка содержит логи выполнения заданий с полным выводом консоли.

## Файлы

- `run_YYYYMMDD_HHMMSS.log` - полный вывод консоли при запуске заданий
- `RESULTS_SUMMARY.md` - итоговый отчет с описанием результатов

## Генерация логов

Логи создаются автоматически при запуске:

```bash
bash run_and_save_logs.sh
```

Скрипт:
1. Запускает все задания последовательно
2. Сохраняет вывод в лог файл
3. Создает итоговый отчет с описанием результатов

## Для преподавателя

Логи включены в Git репозиторий для проверки корректности выполнения заданий.  
Вы можете просмотреть:
- Полный вывод каждого задания
- Метрики производительности
- Сообщения об ошибках (если были)

## Воспроизведение

Для повторного запуска на другой машине:

```bash
# 1. Клонируйте репозиторий
git clone <repo-url>

# 2. Установите окружение
bash setup_env.sh

# 3. Загрузите тестовое видео
bash download_test_video.sh

# 4. Запустите с логированием
bash run_and_save_logs.sh
```
